EquinoxOS Local Web

This page is rendered by htmlview.elf from the EXT2 disk image. It is not a full browser yet, but it proves the first browser layer: reading an HTML file, parsing common tags, wrapping text, and drawing the result in a userspace window.

Supported in this viewer

  • Headings with h1, h2, and h3.
  • Paragraphs, line breaks, and simple wrapping.
  • Lists with li elements.
  • Links are highlighted as text.
  • Inline code blocks use a separate color.

Next target

The next step toward a real browser is an HTTP client. Once EquinoxOS can fetch a plain http:// page over TCP, this renderer can become the display layer for downloaded documents.

Try running: run htmlview.elf index.html
